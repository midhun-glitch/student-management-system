# Student Management System (Python + SQLite)

A simple **menu-driven Student Management System** built using **Python and SQLite**.  
This project demonstrates CRUD operations, database connectivity, and clean program structure, making it ideal for beginners and internship portfolios.

---

## Features
- Add new student records
- View all students
- Update existing student details
- Delete student records
- Persistent data storage using SQLite
- Simple command-line interface

---

## Technologies Used
- Python 3
- SQLite3
- Command Line Interface (CLI)
