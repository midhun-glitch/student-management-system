#  Student Management System (Python + SQLite)

A simple and efficient **command-line Student Management System** built using **Python** and **SQLite**.  
This project demonstrates core backend concepts such as CRUD operations, database connectivity, and modular programming, making it ideal for **internship portfolios and beginner projects**.

---

##  Project Overview

The Student Management System allows users to manage student records through a menu-driven interface.  
All data is stored persistently using an SQLite database.
